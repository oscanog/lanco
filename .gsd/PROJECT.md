# Lancotech Trading Platform

Lancotech is a modern trading education and simulation platform focused on
structured learning and repeated practice.

Users can learn trading, practice strategies, simulate markets, track progress,
improve strategy quality, and analyze mistakes in a guided learning workspace.

Brand direction:

- Modern, professional, premium, minimal, clean.
- Trustworthy, innovative, educational, confident, tech-driven.
- Inspired by Bloomberg, Apple, Stripe, Linear, TradingView, Vercel, Framer,
  and Notion.
- Not flashy, not gambling, not crypto-scam styled.

Core message:

Learn. Practice. Master. Trade with Confidence.

Visual rules:

- Primary background: `#F5F5F5`.
- Primary accent: `#48CFCB`.
- Secondary accent: `#229799`.
- Text: `#424242`.
- Mostly white and light backgrounds.
- Aqua accents only for highlights.
- Lots of whitespace, soft shadows, rounded corners, premium gradients, subtle
  glassmorphism, smooth transitions.
